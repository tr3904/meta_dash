## Meta Ads Performance Analysis Dashboard
## 1) Project Overview
This project presents a comprehensive analysis of Meta (Facebook/Instagram) advertising performance using an interactive dashboard. It focuses on evaluating campaign effectiveness across key metrics such as engagement, conversions, audience demographics, and ad formats.
## 2) Key Features
a) KPI tracking (Impressions, Clicks, CTR, Conversions, Engagement Rate)
b) Audience analysis (Gender & Age segmentation)
c) Geographic performance insights
d) Time-based engagement trends (weekly & hourly)
e) Ad format performance comparison
f) Actionable business recommendations
